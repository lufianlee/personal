## PROFESSIONAL PHILOSOPHY

### Proactive Security 철학
18년간 인프라 및 보안 분야에서 온프레미스부터 클라우드까지 다양한 환경의 보안 이슈를 해결해왔습니다. **사고 발생 후 대응이 아닌 사전 예방 중심의 보안 체계**를 지향하며, 잠재적 위협을 선제적으로 탐지하고 차단하는 Proactive Security 문화를 구축해왔습니다.

단순히 보안 정책을 수립하고 강제하는 것이 아니라, 개발팀과의 긴밀한 협업을 통해 보안을 개발 프로세스에 자연스럽게 통합하고, 이를 통해 조직의 효율성을 높이는 것을 지향합니다. DevSecOps와 자동화를 통해 **보안을 장애물이 아닌 가속기**로 만들며, 보안이 비즈니스 성장을 지원하는 핵심 동력이 되도록 합니다.

### AI 시대의 보안 리더십
**AI For Security**와 **Security For AI**를 선도하며, AI 기술을 활용한 보안 자동화와 생성형 AI 서비스의 안전한 활용을 위한 가이드라인을 구축해왔습니다. 기술과 경험을 공유하고 함께 성장하는 문화를 중요하게 생각하며, AWS Community Builder로서 보안 커뮤니티를 리드하고 있습니다.

앞으로는 **Zero Trust 아키텍처, 클라우드 네이티브 보안, AI 기반 보안** 분야에서 기술 리더십을 발휘하여, 조직의 디지털 트랜스포메이션을 보안 관점에서 선도하고 차세대 보안 인프라를 구축하는 것을 목표로 하고 있습니다.
